# 变更记录

## v1.0.0 (2026-03-26)
- 初始版本
- 颜色反卷积 (H-DAB, H-E, skimage HED)
- H-Score 计算 + 阳性率
- ROI 区域选择
- 批量分析 + CSV 导出
- 评分叠加可视化 (蓝/绿/橙/红)
- 暗色主题 UI
- macOS + Windows 自动构建 (GitHub Actions)

### 修复
- 修复 matplotlib 中文字体乱码
- 修复字符串中中文引号导致的语法错误
- 添加打开文件夹功能
- 按 IHC_Profiler 标准统一灰度区间定义，明确背景为 236-255 并自动排除
- 调整 IHC Score 计算展示，强度评分改为按主导阳性等级判定并统一中文结果文本
- 修复 DAB 通道按单图归一化导致阴性样本被误判为阳性的风险，改为稳定灰度映射
- 批量分析结果和 CSV 导出改为 [图片名称 / 总像素 / 高强阳 / 中阳 / 低阳 / 阴性 / 临床判定 / 强度评分 / 比例评分 / IHC评分] 结构
- 使用桌面 CALD KI67 样例图校正评分规则：临床判定改为按阳性率阈值输出 Positive/Negative
- 强度评分改为基于阳性区域平均灰度，减少弱串色导致的全样本弱阳性偏差
